# mining-pool-list
Curated lists of cryptocurrency mining pools and Stratum endpoints collected from public sources, intended for firewall and network-level blocking.

This repository contains aggregated lists of cryptocurrency mining pools, Stratum servers, and related network endpoints, collected from multiple public and community sources.
**The primary goal** is to support network administrators and security teams in detecting and blocking unauthorized mining activity (cryptomining) at the firewall, DNS, or proxy level.

# mining-pool-list (RU)
